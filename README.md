StableRatioSwap Frontend, see StableRatioSwap Backend repo for getting started instructions

To run this locally, run "npm start" in the root folder. Running it locally will only interact with a mock contract. To test with the "real" contract with external integrations, run "npm run-script deploy", which will deploy a build to the "gh-pages" branch and be useable at https://dhanush123.github.io/stableratioswap-frontend/
